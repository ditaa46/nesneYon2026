# 🖥️ Mini OS + Turkish Programming Language

## 🇹🇷 Türkçe

Bu proje, nesne yönelimli programlama dersi ve veritabanı dersi kapsamında geliştirilmiş basit bir mini işletim sistemi ve Türkçe programlama dili simülasyonudur.

### Özellikler
- Terminal arayüzü
- Basit Türkçe programlama dili (`yaz`, `eger`, `degilse`)
- Dosya düzenleyici (editor)
- Dosya sistemi simülasyonu
- Fotoğraf görüntüleyici (folder + image viewer)
- Basit komutlar:
  - `dir`
  - `run`
  - `edit`

### Amaç
- Yorumlanan dil mantığını anlamak
- Basit bir interpreter geliştirmek
- Dosya sistemi ve UI mantığını simüle etmek

## 🇬🇧 English

This project is a simple mini operating system simulation combined with a custom Turkish-based programming language, developed as part of coursework.

### Features
- Terminal interface
- Custom Turkish-like programming language (`yaz`, `eger`, `degilse`)
- File editor
- Virtual file system
- Folder and image viewer
- Basic commands:
  - `dir`
  - `run`
  - `edit`

### Purpose
- To understand how interpreters work
- To simulate a basic programming language
- To demonstrate file system and UI interaction concepts

## 🚀 How to Run

1. Clone the repository
2. Open `index.html` in a browser
   or use Live Server
3. Use the terminal input to run commands
